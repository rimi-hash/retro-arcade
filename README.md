# Retro Arcade Website

A neon-styled mini arcade built with HTML, CSS, and JavaScript. The project brings together three classic browser games - Tic Tac Toe, Memory Cards, and Rock Paper Scissors - inside one responsive retro interface.

The focus of this project is simple gameplay, clean UI interactions, and a nostalgic arcade feel without using any external game framework.

## Games Included

### Tic Tac Toe

- Two-player game mode
- Requires both player names before the board unlocks
- Shows individual scores for Player X and Player O
- Highlights the winning line
- Includes a Play Again option

### Memory Cards

- Card matching game with 8 pairs
- Requires player name confirmation before playing
- Tracks moves, time, and matched pairs
- Shows a win message after all pairs are matched
- Includes a Play Again option

### Rock Paper Scissors

- Player vs computer
- Requires player name confirmation before playing
- Runs for 5 rounds per match
- Tracks player wins, computer wins, and draws
- Announces the final match result

## Features

- Retro arcade-inspired design
- Neon color palette and glowing UI elements
- Responsive layout for desktop and mobile screens
- Smooth section-based navigation
- Simple name confirmation flow before each game starts
- No backend required
- No build tools required
- Runs directly in the browser
